How does your computer commicate with the Internet?

First, you start with your browser.

When you type a URL into it, and hit enter, you are telling the browser to ask a server for a webpage.

The server starts talking to computer in HTTP, which is the language in which one computer asks another for documents

What transfers between the computer and server is mainly made up of 'GET requests'

What is a website?

  • A collection of files that get excuted in the browser, mainly in HTML, Javavscript, and CSS format
  • HTML is what the browser displays — can be paragraphs, headings, sliders, graphs, images, anything. When developing, it is used as the structure of the website
  • We can see it if we use Developer Tools in our browser
  • However, you can see that the content changes when we interact with the page
  • This is caused by Javascript, which controls the behavior of the HTML
