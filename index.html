Why is the web innovative?

You can share anything with anyone you want to, and you have access to all of the infomation in the world

What are the important components of a website?

Content, images, interactivity

What are the languages used for the web?

  • HTML, which is the structure.
  • CSS, which styles the page.
  • Javascript, which controls the behavior.
HTML is what the browser displays — can be paragraphs, headings, sliders, graphs, images, anything. It is the structure of the website. We can see it if we use Developer Tools in our browser. However, you can see that the content changes when we interact with the page. This is caused by Javascript, which controls the behavior of the HTML. We can see the CSS, the code that styles, in the Styles tab.

Demo

HTML

<h2>Header</h2> <p>a paragraph</p>

CSS

p, h2 { color: red; font-family: Arial; } h2 { background-color: tan; }

Javascript

$("h2").click(function() { $("p").toggle({fade: 200}); });
